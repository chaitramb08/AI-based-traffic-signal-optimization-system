LiveJunction AI

Project Description
An AI-based intelligent traffic signal optimization system designed for mixed traffic conditions. The system analyzes real-time traffic density and dynamically adjusts signal timings to reduce congestion and waiting time.

Features
- Real-time traffic analysis  
- Dynamic signal timing  
- Priority lane detection  
- Smart decision-making  

Technologies Used
- Python  
- Flask  
- HTML  
- CSS  
- JavaScript  

How to Run
1. Run backend.py  
2. Open browser  
3. View dashboard  

Impact
Helps reduce traffic congestion and improves road efficiency.
